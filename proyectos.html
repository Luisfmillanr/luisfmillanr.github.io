Conversor de Monedas

Este proyecto nació de la necesidad de simplificar el proceso de conversión de divisas para usuarios de todo el mundo. Utilicé Java y Maven para desarrollar una aplicación robusta que consume una API de tasas de cambio en tiempo real, brindando resultados precisos y rápidos.

Sistema de Gestión de Inventarios

Diseñado para pequeñas empresas, este sistema fue creado con Python y SQLite para gestionar y optimizar la administración de inventarios. El sistema reduce significativamente el tiempo y el esfuerzo necesarios para controlar el stock y realizar pedidos.
